About this calculator

This scientific calculator supports trigonometric functions (sin, cos, tan), logarithms (log base 10, natural log), square root, powers, factorials, absolute value, and the constants π and e. Toggle between degree and radian mode for trig functions.

Function reference

  • sin / cos / tan — trigonometric functions (use DEG/RAD toggle)
  • log — base-10 logarithm  |  ln — natural logarithm
  • √ — square root  |  x² — square  |  x^y — power
  • x! — factorial (integers 0–20)  |  |x| — absolute value
  • π ≈ 3.14159…  |  e ≈ 2.71828…

FAQ

When should I use DEG vs RAD?
Use DEG if your angles are in degrees (e.g. sin(90°) = 1). Use RAD if your angles are in radians (e.g. sin(π/2) = 1).
How do I enter a power like 2^10?
Type 2, then click x^y (which inserts ^), then 10, then =.
